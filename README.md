# La Casa Oaxaca Website

A Next.js site scaffold for La Casa rental, featuring direct booking, property details, amenities, gallery, and contact pages.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Features

- Landing page with direct booking CTA
- About, amenities, units, gallery, FAQ, and contact pages
- Booking form with a 10% direct discount calculator
- Tailwind CSS styling and responsive layout
